---
title: "Check Domain Status"
api: "GET https://superreseller-qa.nominus.host/forms/api/v2/domains/{domain}/check"
description: "This endpoint provides a command to check if a domain name is available and
        can be registered at the registry."
---

### Path

<ParamField path="domain" type="string" required>
  Domain name to check for availability
</ParamField>

### Response

<ResponseField name="code" type="integer" required>
  Availability code
</ResponseField>

<ResponseField name="description" type="string" required>

Description

</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://superreseller-qa.nominus.host/forms/api/v2/domains/{domain}/check' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "domain": ""
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "code": 210,
  "description": "Available"
}
```

</ResponseExample>
